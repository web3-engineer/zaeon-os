## ğŸ“¦ Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/YOUR_USERNAME/zaeon-os.git](https://github.com/YOUR_USERNAME/zaeon-os.git)
    cd zaeon-os
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Environment Setup:**
    Create a `.env.local` file and add your Wepin keys:
    ```env
    NEXT_PUBLIC_WEPIN_APP_ID=your_app_id
    NEXT_PUBLIC_WEPIN_APP_KEY=your_api_key
    ```
     Google API keys also needed, please go to google console for more info ***
4. 
4.  **Run the application:**
    ```bash
    npm run dev
    ```
