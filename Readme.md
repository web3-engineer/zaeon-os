## ğŸ“¦ Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/YOUR_USERNAME/zaeon-os.git](https://github.com/YOUR_USERNAME/zaeon-os.git)
    cd zaeon-os
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Environment Setup:**
    Create a `.env.local` file and add your keys (Movement RPC, AI Keys, etc.):
    ```env
    NEXT_PUBLIC_MOVEMENT_RPC=https://...
    NEXT_PUBLIC_AI_API_KEY=...
    ```

4.  **Run the application:**
    ```bash
    npm run dev
    ```
